
# Healthko Telehealth Platform

## Overview
Healthko is a modern telehealth platform designed for secure and real-time healthcare consultations between doctors and patients through live video communication.

## Core Features
- Real-time video consultations
- Appointment scheduling
- Secure messaging
- E-prescriptions
- Medical records management
- Notifications and reminders

## Tech Stack
- Frontend: Next.js + React
- Backend: Node.js + Express
- Database: PostgreSQL
- Realtime: Socket.io + WebRTC
- Storage: AWS S3

## Dashboards
- Doctor Dashboard
- Patient Dashboard
- Admin Dashboard

## Main Workflow
1. Patient books appointment
2. Doctor confirms appointment
3. Patient joins consultation room
4. Doctor starts video consultation
5. Notes and prescriptions are generated
6. Consultation history is stored
